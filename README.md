# MaVionix Digital Services Website

A modern responsive company landing page created for the MaVionix Web Development Internship Assignment.

## Features

- Responsive design
- Animated hero section
- Services cards
- Portfolio section
- Contact form validation
- Dark theme toggle
- Framer Motion animations
- Back-to-top button
- Responsive mobile menu
- SEO title and meta tags
- Clean React component structure

## Tech Stack

- React.js
- Tailwind CSS
- Framer Motion
- Lucide React
- Vite
- Vercel

## Sections

- Hero
- About
- Services
- Portfolio
- Contact
- Footer
